This program lets your twitch chat control your keyboard while streaming.
Setup:
  1) Use this link to get the token: https://twitchapps.com/tmi/
  2) Put your channel name and token in code
  3) Start streaming with program like OBS

You can modify oneButton and twoButton variables so.
you can choose what buttons you want to be pressed by chat.
and for how long they will be pressed (in seconds).
At twoButton function the second (additional) button will only. 
be pressed slightly (if you turn a car, for exaple), this can also be modified.
Can support 5 chat commands at the same time.
