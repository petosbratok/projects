Python 3.7, bs4 required.
This project was inspired and provided by jetbrains academy, but I changed some parts of it so it works better.
All the code is writted by me.
Program is a simple browser without graphics and css support.
It is best used for simple websites like wikipedia.
Not really usable in real life conditions but was fun coding it.

How to use:
  1) Type the link (including "https://") and press enter.
  2) To exit type "exit".
  3) You can go back to previous page by typing "back".
