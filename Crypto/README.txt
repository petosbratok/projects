Eng

To be used with Python 3.7.
This program uses Caesar cipther to encrypt .txt files.
Folder has test.txt file so you can try it out.
Characters supported by the program:

 !"#$%&\'()*+,-./0123456789:;<=>?@ABCDEFGHIJKLMNO
PQRSTUVWXYZ[\\]^_`abcdefghijklmnopqrstuvwxyz{|}~

Those are characters 32-126 from utf-8.
If your .txt contains other characters, you have to modify the program or
you will lose your files.

Rus

В папке лежит .txt файл, чтобы вы смогли ее протестировать.
Символы, поддериживаемые программой:

 !"#$%&\'()*+,-./0123456789:;<=>?@ABCDEFGHIJKLMNO
PQRSTUVWXYZ[\\]^_`abcdefghijklmnopqrstuvwxyz{|}~


Это символы 32-126 из utf-8.
Если ваш .txt содержит другие символы, вы должны изменить программу или
вы потеряете свои файлы.
